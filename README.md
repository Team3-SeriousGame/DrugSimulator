* 마약 섭취량이 증가한 경우를 레벨로 표현
* 메시지 알림으로 간단한 미션 알려주고, 미션 성공 시 상단 Todo list 체크표시
* 미션은 상단 타이머가 끝나기 전까지 수행해야 하며, 끝나면 게임 종료. 마찬가지로 자동차와 부딪히면 HP 감소(0일 경우 게임 종료)
* Light와 Audio 디자인
